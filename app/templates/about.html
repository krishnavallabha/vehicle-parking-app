About Slotly

Slotly is a smart, user-centric vehicle parking management system built to simplify the way we find and reserve parking spaces. Whether youâ€™re a commuter searching for a nearby available spot or an administrator overseeing multiple parking lots, Slotly offers a streamlined experience tailored to your needs.

Designed with modern urban challenges in mind, Slotly automates and optimizes parking through intelligent features including:

  • Real-time spot availability tracking
  • Instant reservations with auto-allocation of parking spots
  • Role-based dashboards for both users and administrators
  • Secure user authentication and session handling
  • Comprehensive lot and reservation management

Our goal is to reduce congestion, improve parking efficiency, and eliminate the daily frustration of hunting for a spot all through a clean, responsive, and intuitive interface.

Project Background

This application is a college-level software engineering project developed as part of the Modern Application Development I course for the Bachelor of Science in Data Science and Applications program. It was conceptualized, designed, and implemented to demonstrate proficiency in full-stack development using technologies such as:

  • Flask for backend development
  • SQLite for lightweight, embedded database storage
  • Jinja2 templating for dynamic HTML rendering
  • HTML, CSS, and Bootstrap for frontend interface

Every component from database schema design to user flow and role-based logic was carefully architected to reflect real world application development practices, making Slotly not just a project, but a complete working prototype of a scalable parking solution.
